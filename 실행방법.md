# 실행 방법 — Secret Scanner

브라우저에서 직접 돌려보는 방법 정리. (나중에 README.md 에 합칠 예정)

## 사전 요구사항
- Node.js 18+ (권장 20 / 22)
- Docker 실행 시: Docker Desktop

---

## 방법 1. 로컬 개발 모드 (가장 빠름, 권장)

프론트엔드와 백엔드를 각각 띄운다. 코드 수정 시 자동 반영(HMR).

**터미널 1 — 백엔드**
```bash
cd secret-scanner/backend
npm install        # 최초 1회만
npm run dev        # http://localhost:4000
```

**터미널 2 — 프론트엔드**
```bash
cd secret-scanner/frontend
npm install        # 최초 1회만
npm run dev        # http://localhost:5173
```

→ 브라우저에서 **http://localhost:5173** 열기
- `/api` 요청은 Vite 프록시가 자동으로 `:4000`(백엔드)로 넘긴다.
- **"예제 체험"** 버튼을 누르면 바로 탐지 결과(해설 카드 포함)를 볼 수 있다.

---

## 방법 2. 프론트엔드 빌드 미리보기

프로덕션 빌드 결과를 확인한다. (해설 카드 데이터를 보려면 백엔드도 따로 떠 있어야 함)
```bash
cd secret-scanner/frontend
npm run build      # dist/ 생성
npm run preview    # http://localhost:4173
```

---

## 방법 3. Docker 전체 스택 (Nginx + Express + PostgreSQL)

실제 배포와 동일한 구성. Nginx가 정적 파일 서빙 + `/api` 프록시를 담당한다.
```bash
cd secret-scanner
cp .env.example .env          # 최초 1회 (필요 시 AI 키 입력)
docker compose up --build     # http://localhost:8080
```

→ 브라우저에서 **http://localhost:8080** 열기

**종료 / 정리**
```bash
docker compose down           # 중지
docker compose down -v        # DB 볼륨까지 삭제
docker compose logs -f backend   # 백엔드 로그 보기
```
